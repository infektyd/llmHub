# Workspace Directory

**Purpose**: Development artifacts and temporary files for AI brainstorming sessions.

## Files

### `xcode-console-dump.txt`
Raw Xcode console output for sharing with AI models.

**Usage:**
1. Run your app in Xcode
2. Copy console output (Cmd+A, Cmd+C in Console pane)
3. Paste into this file (overwrites previous content)
4. Share with AI for debugging/analysis

**Format:** Plain text with all Xcode metadata preserved (timestamps, process IDs, log levels, etc.)

**Git Status:** Ignored (not committed to repository)
